# Technology Stack - CIMA (Real Estate Platform)

## Core Technologies
- **Language:** C# (.NET 9)
- **Application Framework:** ABP Framework 9.3 (Layered Architecture)
- **Data Access:** Entity Framework Core
- **Database:** PostgreSQL

## Frontend & UI
- **Frontend Framework:** Blazor WebApp (.NET 9)
- **Execution Model:** Unified hosting model (Hybrid SSR/Interactive) to balance performance, SEO, and security.
- **Styling:** Tailwind CSS

## Infrastructure & DevOps
- **Containerization:** Docker (with Docker Compose for local dev)
- **Hosting:** Railway (Staging environment)
- **CI/CD:** GitHub Actions for automated build, test, and deployment workflows.
